# 🚀 Day 10 – JavaScript DOM & Events

## Topics Covered

* Introduction to the Document Object Model (DOM)
* Selecting HTML elements using `getElementById()` and `querySelector()`
* Handling user interactions with `onclick` and `addEventListener()`
* Preventing default form submission using `event.preventDefault()`
* Reading user input using `.value`
* Form validation for text fields and radio buttons
* Updating page content using `innerText`
* Modifying element styles dynamically
* Disabling elements using `.disabled`
* Password length counter
* Password strength indicator
* Creating new HTML elements dynamically
* Appending elements to the DOM using `appendChild()`

---

## Selecting DOM Elements

Used to access HTML elements for manipulation.

```javascript
let btn = document.getElementById("signupbtn");
let name = document.getElementById("text");
let email = document.getElementById("email");
```

---

## Handling Click Events

Executed when the Sign Up button is clicked.

```javascript
btn.onclick = function(event){
    event.preventDefault();
}
```

---

## Reading User Input

Retrieve values entered by the user.

```javascript
let uName = name.value;
let uEmail = email.value;
let uPwd = pwd.value;
```

---

## Form Validation

Checked for empty input fields and user type selection.

```javascript
if(uName === "" || uEmail === "" || uPwd === ""){
    message.innerText = "Please fill all the fields.";
}
else if(user == null){
    message.innerText = "Please select the user type.";
}
```

---

## Updating DOM Content

Changed page content dynamically after successful validation.

```javascript
heading.innerText = "Welcome " + uName + "!";
message.innerText = "Account created successfully.";
```

---

## Dynamic Styling

Modified CSS properties using JavaScript.

```javascript
heading.style.color = "white";
message.style.color = "green";
```

---

## Button State

Disabled the button after successful registration.

```javascript
btn.innerText = "Registered ✓";
btn.disabled = true;
```

---

## Input Event

Executed continuously while typing.

```javascript
pwd.addEventListener("input", function(){

});
```

---

## Password Character Counter

Displayed the password length in real time.

```javascript
pwdCount.innerText = pwd.value.length;
```

---

## Password Strength Meter

Evaluated password strength based on its length.

```javascript
if(len < 6){
    strength.innerText = "🔴 Weak";
}
else if(len < 10){
    strength.innerText = "🟡 Medium";
}
else{
    strength.innerText = "🟢 Strong";
}
```

---

## Creating Elements Dynamically

Created new HTML elements using the DOM.

```javascript
let p = document.createElement("p");
```

---

## Updating Element Content

Assigned text to newly created elements.

```javascript
p.innerText = "To create click on yellow colored create button on the top";
```

---

## Appending Elements

Inserted dynamically created elements into the webpage.

```javascript
document.body.appendChild(p);
```
