## `for` Loop

Used when the number of iterations is known.

```javascript
for(let i = 0; i < characters.length; i++){
    console.log(characters[i]);
}
// Output:
// Lu
// Shiyi
// Xu
// Bai
// lusi
```

---

## `for...in`

Iterates through object keys.

```javascript
for(let i in unknown){
    console.log(unknown[i]);
}
// Output:
// ['lu','shi']
// [false,false]
// [Function: never]
```

---

## `for...of`

Iterates through iterable values like arrays.

```javascript
for(let i of house){
    console.log("Houses:", i);
}
// Output:
// Houses: slytheriin
// Houses: Gryfindore
// Houses: Hufflepuff
```

---

## Map

Stores data as key-value pairs.

```javascript
const m = new Map();
m.set("Toko","Cosmic writer");
m.set("Yuzhou","Philosophical writer");

for(let w of m){
    console.log(w);
}
// Output:
// ['Toko','Cosmic writer']
// ['Yuzhou','Philosophical writer']
```

---

## `forEach()`

Executes a callback function for every array element.

```javascript
house.forEach(function(h){
    console.log(h);
});
// Output:
// slytheriin
// Gryfindore
// Hufflepuff
```

## `if...else`

Executes code based on conditions.

```javascript
if(number % 2 == 0){
    console.log("Even");
}else{
    console.log("Odd");
}
// Output:
// Even / Odd
```
