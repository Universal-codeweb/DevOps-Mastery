# Day 7 – JavaScript Functions

**Date:** 23-06-2026

## Overview

Learned how JavaScript functions help organize, reuse, and execute code efficiently. Explored different types of functions, parameters, return values, scope, callbacks, constructors, asynchronous functions, and modern ES6 features.

---

## Topics Covered

### Function Declaration

Created reusable functions using the `function` keyword.

```javascript
function sayHello() {
    console.log("Hello Universe");
}
```

Used to execute a specific task whenever needed.

---

### Function Scope

Learned how functions can access variables from their outer scope.

```javascript
let mess = "Hello Universe";

function sayHello() {
    console.log(mess);
}
```

The function accesses the global variable `mess`.

---

### Named Function Expression

```javascript
let namedFunction = function sum(n1, n2) {
    console.log(n1 + n2);
};
```

A function stored inside a variable while also having its own name.

---

### Anonymous Function

```javascript
let anonymousFunction = function(p, str) {
    let res = p.test(str);
    let matches = str.match(p);
    console.log(res, matches);
};
```

Functions without a direct name, commonly used as callbacks and expressions.

---

### Arrow Functions

```javascript
let say = () => {
    console.log("Welcome to this universe filled with mysteries");
};
```

Modern and concise syntax for creating functions.

---

### Immediately Invoked Function Expression (IIFE)

```javascript
(function () {
    console.log("Level 5");
})();
```

Executes immediately after being created.

---

### Parameters and Arguments

Parameters receive values passed during function calls.

```javascript
function greeting(name, message = "Hello") {
    console.log(message, name);
}

greeting("Universe");
```

Learned:

* Parameters
* Arguments
* Default Parameters

---

### Return Values

Functions can return values for further use.

```javascript
function mul(n, callBack) {
    return callBack(n);
}

const cal = (n) => n * n;
```

The callback function returns the square of a number.

---

### Callback Functions

A function passed as an argument to another function.

```javascript
function mul(n, callBack) {
    return callBack(n);
}
```

Used extensively in:

* Event Handling
* APIs
* React
* Node.js

---

### Rest Parameters

Allows functions to accept multiple values.

```javascript
function contactUniversians(message, ...names) {
    names.forEach(name =>
        console.log(message, name)
    );
}
```

Used for handling unlimited arguments.

---

### Constructor Functions

Used to create multiple objects with the same structure.

```javascript
function Universe(name, year) {
    this.name = name;
    this.year = year;
}

const enter = new Universe("Zverse", 130);
```

Learned:

* Constructor Functions
* `new` Keyword
* `this` Keyword

---

### Async Functions

Functions that work with asynchronous operations and return Promises.

```javascript
async function create() {
    let hell = true;
    return hell;
}
```

Consumed using:

```javascript
create().then(result =>
    console.log(result)
);
```

---

### Functions with Regular Expressions

Used functions to perform pattern matching.

```javascript
anonymousFunction(/hello/i, "Hello Universe");
```

Learned:

* `test()`
* `match()`
* Regex Flags (`i`)
* Pattern Matching
