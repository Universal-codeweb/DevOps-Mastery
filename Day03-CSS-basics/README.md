## Key Learnings

* CSS controls the presentation layer of a webpage.
* Classes are reusable while IDs are unique.
* Every HTML element follows the CSS Box Model.
* Padding affects internal spacing, while margin affects external spacing.
* Hover effects improve user interaction and user experience.
* Navigation bars are essential for website navigation and accessibility.
