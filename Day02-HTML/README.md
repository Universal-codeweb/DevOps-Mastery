## Practical Work

Created a mini website called "Tekrearc" featuring:

* Header and Navigation
* About Section
* Image Display
* Project Status Table
* Feedback Form
* Login Section
* Sign Up Navigation

## Key Learnings

* HTML provides the structure of a webpage.
* Tags define elements on a webpage.
* Attributes provide additional information to elements.
* Forms collect user input.
* Semantic tags improve readability and accessibility.
* Links can navigate within a page or to another page.
