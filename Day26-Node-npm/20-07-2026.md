# 🌐 What is Node.js?

Node.js is a **JavaScript Runtime Environment** built on Google's **V8 JavaScript Engine**. It allows JavaScript to run outside the browser, making it possible to build backend applications, APIs, CLI tools, and scalable server-side software.

### Learned

- What Node.js is
- Why Node.js was created
- Difference between JavaScript and Node.js
- Runtime Environment
- V8 JavaScript Engine
- How Node.js executes JavaScript
- Node.js architecture

### Execution Flow

```text
JavaScript Code
      │
      ▼
Node.js Runtime
      │
      ▼
V8 Engine
      │
      ▼
Machine Code
      │
      ▼
Operating System
```

# 📂 File System (fs) Module

The **fs** module is a built-in Node.js module that allows applications to interact with files and folders.

### Practiced

- Reading files
- Writing files
- Appending data
- Renaming files
- Deleting files
- Error handling
- Synchronous vs Asynchronous operations

### Reading a File

```javascript
const fs = require("fs");

fs.readFile("message.txt", "utf8", (err, data) => {
    if (err) {
        console.log(err);
        return;
    }

    console.log(data);
});
```

### Writing a File

```javascript
const fs = require("fs");

fs.writeFile("notes.txt", "Learning Node.js", (err) => {
    if (err) {
        console.log(err);
        return;
    }

    console.log("File created successfully!");
});
```

### Appending Data

```javascript
fs.appendFile("notes.txt", "\nLearning NestJS", (err) => {
    if (err) {
        console.log(err);
        return;
    }

    console.log("Content appended!");
});
```

# 📦 Modules in Node.js

Modules help organize applications into smaller, reusable files.

### Learned

- What is a module?
- Why modules are important
- Exporting modules
- Importing modules
- CommonJS
- ES Modules
- Built-in modules
- Third-party modules

### Exporting

```javascript
function add(a, b) {
    return a + b;
}

module.exports = add;
```

### Importing

```javascript
const add = require("./math");

console.log(add(5, 8));
```

### CommonJS vs ES Modules

**CommonJS**

```javascript
const fs = require("fs");

module.exports = add;
```

**ES Modules**

```javascript
export function add(a, b) {
    return a + b;
}

import { add } from "./math.js";
```

# ⚙️ npm Scripts

Created custom scripts to simplify running applications.
