# Day 14 – TypeScript Introduction

## Overview

Learned the fundamentals of TypeScript, including installation, compiler setup, writing TypeScript code, compiling it into JavaScript, and running it with Node.js.

## Installation & Setup

Install the TypeScript compiler globally:

```bash
npm install -g typescript
```

Verify the installation:

```bash
tsc -v
```

Compile a TypeScript file:

```bash
tsc intro.ts
```

Run the generated JavaScript file:

```bash
node intro.js
```

## Topics Covered

* Introduction to TypeScript

## Practice

### Greeting Function

Created a function with a `void` return type to print a welcome message.

```ts
function greet(): void {
    console.log("Welcome");
}
```

### Variables

Declared variables using explicit TypeScript data types.

```ts
let userName: string = "Swetha";
let userAge: number = 18;
let isGraduated: boolean = false;
```

### Array

Created an array that accepts only string values.

```ts
let skills: string[] = ["HTML", "CSS", "JavaScript"];
```

### Console Output

Displayed variable values and array contents using `console.log()`.

## Common Errors Practiced

* Variable name conflicts (`name`)
* Type mismatch (`string` assigned to `number`)
* Misspelled type names (`sting` → `string`)
