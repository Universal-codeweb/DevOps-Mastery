# Async/Await Concepts

## 1. Async Function

```javascript
async function functionName() {

}
```

- The `async` keyword makes a function asynchronous.
- It allows the use of the `await` keyword inside the function.
- An `async` function always returns a Promise.

---

## 2. Await Keyword

```javascript
const result = await promise;
```

- `await` pauses only the current async function until the Promise is resolved or rejected.
- Once the Promise is fulfilled, the resolved value is returned.

---

## 3. Sequential Execution

Using `await` allows asynchronous tasks to execute one after another.

```text
Task 1
   ↓
Task 2
   ↓
Task 3
```

Each task starts only after the previous Promise has completed.

---

## 4. Error Handling

```javascript
try {

}
catch(error){

}
```

- `try` contains the asynchronous code.
- If any Promise rejects, execution jumps directly to the `catch` block.
- This prevents the application from crashing.

# Fetch API Practice

# Fetch API Code Explanation

## 1. Getting User Input

```javascript
const pokename = document
    .getElementById("txt")
    .value
    .toLowerCase();
```

- Retrieves the Pokémon name entered by the user.
- Converts it to lowercase because the PokéAPI expects lowercase names.

---

## 2. Sending a GET Request

```javascript
const response = await fetch(
    `https://pokeapi.co/api/v2/pokemon/${pokename}`
);
```

- Sends an HTTP GET request to the PokéAPI.
- `fetch()` returns a Promise.
- `await` waits until the server responds.

---

## 3. Checking the Response

```javascript
if (!response.ok) {
    throw new Error("Couldn't fetch resource");
}
```

- `response.ok` is `true` for successful responses.
- If the request fails (404, 500, etc.), an error is thrown.

---

## 4. Converting JSON to JavaScript Object

```javascript
const data = await response.json();
```

- Converts the JSON response into a JavaScript object.
- `response.json()` also returns a Promise, so `await` is required.

---

## 5. Accessing the Sprite

```javascript
const pokemonSprite =
    data.sprites.front_default;
```

Extracts the Pokémon sprite image URL from the API response.

---

## 6. Updating the DOM

```javascript
image.src = pokemonSprite;
image.style.display = "block";
```

- Sets the fetched image as the `<img>` source.
- Makes the image visible on the webpage.

---

## 7. Error Handling

```javascript
catch(error){
    console.error(error);
}
```

Handles network errors or invalid Pokémon names without stopping the application.
