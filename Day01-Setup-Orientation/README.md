## Key Concepts Learned

### Frontend

The frontend is the part of a website that users can see and interact with. It is built using HTML, CSS, and JavaScript.

### Backend

The backend handles business logic, databases, authentication, and APIs. Users do not directly see the backend.

### HTML

HTML provides the structure of a webpage.

### CSS

CSS controls the appearance and styling of webpage elements.

### JavaScript

JavaScript adds interactivity and dynamic behavior to webpages.

### Node.js

Node.js allows JavaScript to run outside the browser.

### Git

Git is a version control system used to track code changes.

### GitHub

GitHub is a cloud platform used to host Git repositories.
