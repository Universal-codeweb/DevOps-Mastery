# Day 6 – JavaScript Variables & Types
**Date:** 22-06-2026

## Overview
Learned the fundamentals of JavaScript including variables, data types, operators, objects, the `this` keyword, and regular expressions.

---

## Variables
Variables are used to store data in memory.

### let
Used for values that can be changed later.

```javascript
let name = "Universian";
```

### const
Used for values that should remain constant.

```javascript
const title = "Cosmic";
```

---

## Data Types

### Number
Stores numeric values.

```javascript
let n1 = 5;
let n2 = 6;
```

### String
Stores text data.

```javascript
let s1 = "Hello, Universe!";
let s2 = 'Single quotes also work';
```

### Boolean
Stores logical values.

```javascript
let isThisWork = true;
const isThisChangable = false;
```

### Null
Represents an intentionally empty value.

```javascript
let age = null;
```

### Symbol
Creates unique identifiers.

```javascript
let sy1 = Symbol("Geeks");
let sy2 = Symbol("Geeks");
```

---

## Arrays
Used to store multiple values in a single variable.

```javascript
const galaxies = ["Andromeda", "Milky Way", "Tone"];
```

---

## Comparison Operators

### Equal (`==`)
Compares only values.

```javascript
5 == "5" // true
```

### Strict Equal (`===`)
Compares value and data type.

```javascript
5 === "5" // false
```

### Not Equal (`!=`)

```javascript
5 != "5" // false
```

### Strict Not Equal (`!==`)

```javascript
5 !== "5" // true
```

## Objects

Objects store data as key-value pairs.

```javascript
const universe = {
  name: "Zverse"
};
```

---

## `this` Keyword

Refers to the current object.

```javascript
const universe = {
  name: "Zverse",
  activate() {
    return `Started evolving, ${this.name}`;
  }
};
```

Output:

```javascript
Started evolving, Zverse
```

---

## Regular Expressions (Regex)

Regex is used for pattern matching and text searching.

```javascript
let pattern = /hello/;
```

### test()

Checks whether a pattern exists in a string.

```javascript
pattern.test("Hello Universe");
```

---

## Regex Flags

### `i` Flag (Case Insensitive)

```javascript
let pattern = /universe/i;
```

Matches:

```text
universe
Universe
UNIVERSE
```

without considering letter case.
