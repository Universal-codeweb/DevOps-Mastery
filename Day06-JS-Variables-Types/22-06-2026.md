# Day 6 – JavaScript Variables & Types
**Date:** 22-06-2026

## Overview
Learned the fundamentals of JavaScript including variables, data types, operators, objects, the `this` keyword, and regular expressions.

---

## Variables
Variables are used to store data in memory.

### let
Used for values that can be changed later.

```javascript
let name = "Universian";
```

### const
Used for values that should remain constant.

```javascript
const title = "Cosmic";
```

---

## Data Types

### Number
Stores numeric values.

```javascript
let n1 = 5;
let n2 = 6;
```

### String
Stores text data.

```javascript
let s1 = "Hello, Universe!";
let s2 = 'Single quotes also work';
```

### Boolean
Stores logical values.

```javascript
let isThisWork = true;
const isThisChangable = false;
```

### Null
Represents an intentionally empty value.

```javascript
let age = null;
```

### Symbol
Creates unique identifiers.

```javascript
let sy1 = Symbol("Geeks");
let sy2 = Symbol("Geeks");
```

---

## Arrays
Used to store multiple values in a single variable.

```javascript
const galaxies = ["Andromeda", "Milky Way", "Tone"];
```

## 3. JavaScript Operators

Learned various types of JavaScript operators and their practical usage.

### Arithmetic Operators

Used for mathematical calculations.

* `+` Addition
* `-` Subtraction
* `*` Multiplication
* `/` Division
* `%` Modulus
* `**` Exponentiation
* `++` Increment
* `--` Decrement

### Assignment Operators

Used to assign and update values.

* `=`
* `+=`
* `-=`
* `*=`
* `/=`
* `%=`
* `**=`

### Comparison Operators

Used to compare values.

* `==`
* `===`
* `!=`
* `!==`
* `>`
* `<`
* `>=`
* `<=`

### Logical Operators

Used to combine conditions.

* `&&` (AND)
* `||` (OR)
* `!` (NOT)

### Bitwise Operators

Perform operations on binary representations.

* `&`
* `|`
* `^`
* `~`
* `<<`
* `>>`
* `>>>`

### String Operators

Used for string concatenation.

```javascript
"Hello" + " Universe"
```

### Conditional (Ternary) Operator

```javascript
condition ? value1 : value2
```

## 4. Expressions

An expression is any valid combination of values, variables, and operators that produces a result.

## 5. Objects

Objects store related data using key-value pairs.

```javascript
const universe = {
  name: "Zverse"
};
```

### Object Method

```javascript
const universe = {
  name: "Zverse",
  activate() {
    return "Started evolving";
  }
};
```

---

## 6. `this` Keyword

`this` refers to the current object.

```javascript
const universe = {
  name: "Zverse",

  activate() {
    return `Started evolving, ${this.name}`;
  }
};
```

Output:

```javascript
Started evolving, Zverse
```

`this.name` refers to `universe.name`.

## 7. Regular Expressions (Regex)

Regex is used for searching, matching, and validating text patterns.

### Creating a Regex

```javascript
let pattern = /hello/;
```

### Testing a Pattern

```javascript
pattern.test("hello universe");
```

Returns:

```javascript
true
```

---

## 8. Regex Flags

### `i` – Case Insensitive

```javascript
/universe/i
```

Matches:

```text
universe
Universe
UNIVERSE
```

### Common Flags

| Flag | Meaning          |
| ---- | ---------------- |
| `i`  | Ignore case      |
| `g`  | Global search    |
| `m`  | Multiline search |

---

## 9. Regex Metacharacters

Special characters with unique meanings.

| Character | Meaning                      |
| --------- | ---------------------------- |
| `.`       | Any character except newline |
| `^`       | Start of string              |
| `$`       | End of string                |
| `\d`      | Any digit (0-9)              |
| `\D`      | Non-digit                    |
| `\w`      | Word character               |
| `\W`      | Non-word character           |
| `\s`      | Whitespace                   |
| `\S`      | Non-whitespace               |

Examples:

```javascript
/\d/
```

Matches:

```text
1 2 3 4 5
```

---

## 10. Character Classes (Brackets)

### Match Specific Characters

```javascript
/[abc]/
```

Matches:

```text
a
b
c
```

### Match Range

```javascript
/[a-z]/
```

Lowercase letters.

```javascript
/[A-Z]/
```

Uppercase letters.

```javascript
/[0-9]/
```

Digits.

### Negation

```javascript
/[^0-9]/
```

Matches everything except digits.

---

## 11. Quantifiers

Specify how many times a pattern can occur.

### `*`

Zero or more times.

```javascript
/a*/
```

### `+`

One or more times.

```javascript
/a+/
```

### `?`

Zero or one time.

```javascript
/a?/
```

### Exact Count

```javascript
/a{3}/
```

Exactly 3 times.

### Range

```javascript
/a{2,5}/
```

Between 2 and 5 times.

### Minimum Count

```javascript
/a{2,}/
```

2 or more times.

---

## 12. Practical Regex Examples

### Email Validation

```javascript
/^\w+@\w+\.\w+$/
```

### Mobile Number

```javascript
/^[0-9]{10}$/
```

### Password Pattern

```javascript
/^(?=.*[A-Z])(?=.*\d).{8,}$/
```

Checks:

* At least one uppercase letter
* At least one digit
* Minimum 8 characters
