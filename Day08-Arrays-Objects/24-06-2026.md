To understand JavaScript Arrays, perform array operations, and explore commonly used Array Methods and Static Array Methods.

## 📚 Topics Covered

### 1. Array Creation

* Array Literal (`[]`)
* Array Constructor (`new Array()`)

### 2. Array Properties

* `length`

### 3. Array Operations

* Accessing Elements using Index
* Adding Elements (`push()`, `unshift()`)
* Removing Elements (`pop()`, `shift()`)

### 4. Array Destructuring

* Extracting values into variables
* Swapping variables using destructuring

### 5. Nested Arrays

* Accessing arrays within arrays

### 6. Array Merging

* Spread Operator (`...`)
* `concat()`

## 🔧 Array Methods Practiced

| Method       | Purpose                           |
| ------------ | --------------------------------- |
| `push()`     | Add element(s) to the end         |
| `pop()`      | Remove the last element           |
| `unshift()`  | Add element(s) to the beginning   |
| `shift()`    | Remove the first element          |
| `slice()`    | Create a shallow copy of an array |
| `concat()`   | Merge arrays                      |
| `join()`     | Convert array into a string       |
| `fill()`     | Fill array with a static value    |
| `includes()` | Check if an element exists        |
| `indexOf()`  | Find the position of an element   |

## 🏛️ Static Array Methods Practiced

Static methods belong to the `Array` object itself and are called using `Array.method()`.

| Static Method     | Purpose                            |
| ----------------- | ---------------------------------- |
| `Array.isArray()` | Checks whether a value is an array |

### Example

```javascript
Array.isArray([1, 2, 3]); // true
Array.isArray("Hello");   // false
Array.isArray({});        // false
```

# Objects & Properties

## Object Creation

Learned how to create JavaScript objects using key-value pairs to represent related data and behaviors.

```javascript
let detail = {
    name: "ne3",
    type: "nebulae",
    age: 1500
};
```

---

## Object Methods

Added functions inside an object to define its behavior and accessed object properties using the `this` keyword.

```javascript
say() {
    alert("Hello, " + this.name);
}
```

---

## Adding Properties

Learned multiple ways to add new properties to an object.

### During Object Creation

```javascript
let detail = {
    isAvailable: true
};
```

### Using Dot Notation

```javascript
detail.year = 2050;
```

---

## Updating Properties

Modified existing property values without recreating the object.

```javascript
detail.age = 1600;
```

---

## Deleting Properties

Removed unnecessary properties using the `delete` operator.

```javascript
delete detail.isAvailable;
```

---

## Dot Notation vs Bracket Notation

Used dot notation for regular property names and bracket notation for property names containing spaces or special characters.

```javascript
planets.name;
planets["moon with water"];
```

---

## Computed Property Names

Created dynamic property names using variables inside square brackets.

```javascript
let galaxy = "andro";

let unknown = {
    [galaxy]: "Lu"
};
```

---

## Method Shorthand

Defined object methods using the ES6 shorthand syntax instead of the traditional function keyword.

```javascript
never() {
    return "nevermore";
}
```

---

## Object Spread Operator

Created a shallow copy of an object using the spread operator.

```javascript
let unknown2 = { ...unknown };
```

---

## Object Destructuring

Extracted object property values into individual variables using destructuring assignment.

```javascript
const { name, year } = universe;
```

---

## Concepts Learned

* Creating objects using object literals.
* Storing properties and methods inside objects.
* Accessing properties using dot and bracket notation.
* Adding, updating, and deleting object properties.
* Using computed property names.
* Defining methods with shorthand syntax.
* Copying objects using the spread operator.
* Extracting values using object destructuring.
