# 🚀 Day 8 - JavaScript Arrays & Methods

## 📌 Objective

To understand JavaScript Arrays, perform array operations, and explore commonly used Array Methods and Static Array Methods.

## 📚 Topics Covered

### 1. Array Creation

* Array Literal (`[]`)
* Array Constructor (`new Array()`)

### 2. Array Properties

* `length`

### 3. Array Operations

* Accessing Elements using Index
* Adding Elements (`push()`, `unshift()`)
* Removing Elements (`pop()`, `shift()`)
* Copying Arrays (`slice()`)

### 4. Array Destructuring

* Extracting values into variables
* Swapping variables using destructuring

### 5. Nested Arrays

* Accessing arrays within arrays

### 6. Array Merging

* Spread Operator (`...`)
* `concat()`

## 🔧 Array Methods Practiced

| Method       | Purpose                           |
| ------------ | --------------------------------- |
| `push()`     | Add element(s) to the end         |
| `pop()`      | Remove the last element           |
| `unshift()`  | Add element(s) to the beginning   |
| `shift()`    | Remove the first element          |
| `slice()`    | Create a shallow copy of an array |
| `concat()`   | Merge arrays                      |
| `join()`     | Convert array into a string       |
| `fill()`     | Fill array with a static value    |
| `includes()` | Check if an element exists        |
| `indexOf()`  | Find the position of an element   |

## 🏛️ Static Array Methods Practiced

Static methods belong to the `Array` object itself and are called using `Array.method()`.

| Static Method     | Purpose                            |
| ----------------- | ---------------------------------- |
| `Array.isArray()` | Checks whether a value is an array |

### Example

```javascript
Array.isArray([1, 2, 3]); // true
Array.isArray("Hello");   // false
Array.isArray({});        // false
```
