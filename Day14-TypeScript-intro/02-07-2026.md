# Day 14 – TypeScript Introduction

## Overview

This program demonstrates the fundamentals of TypeScript, including variable declarations with type annotations, arrays, functions, and console output.

## Concepts Covered

* TypeScript setup and compilation
* Static typing
* Type annotations
* Variables (`string`, `number`, `boolean`)
* Arrays
* Functions
* Console output

## Code Explanation

### Function

```ts
function greet(): void {
    console.log("Welcome");
}
```

* Defines a function named `greet`.
* The return type `void` indicates that the function does not return any value.
* Prints **"Welcome"** to the console.

---

### Calling the Function

```ts
greet();
```

* Invokes the `greet()` function.
* Displays **Welcome** before the remaining output.

---

### String Variable

```ts
let userName: string = "Swetha";
```

* Declares a variable named `userName`.
* Stores a text value.
* `string` ensures only text values can be assigned.

---

### Number Variable

```ts
let userAge: number = 18;
```

* Stores a numeric value.
* `number` prevents assigning non-numeric values.

---

### Boolean Variable

```ts
let isGraduated: boolean = false;
```

* Stores a boolean value (`true` or `false`).

---

### Printing Variables

```ts
console.log("Name:", userName);
console.log("Age:", userAge);
console.log("Graduated:", isGraduated);
```

* Displays the values stored in the variables.

---

### String Array

```ts
let skills: string[] = ["HTML", "CSS", "JavaScript"];
```

* Creates an array containing multiple strings.
* Only string values are allowed.

---

### Printing the Array

```ts
console.log(skills);
```

* Displays all elements of the array.

## Expected Output

```text
Welcome
Name: Swetha
Age: 18
Graduated: false
[ 'HTML', 'CSS', 'JavaScript' ]
```
